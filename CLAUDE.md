## Purpose

This repo (working title "Bike Buddy") is for a simple tool that extracts information about upcoming UCI road racing events and publishes the next event with preferred riders in it.

### Principles

- Use only what we need; nothing more. Don't import libraries when there's an easier way.
- Keep it simple, really simple.
- The shell is your friend.

## Constraints

- Make the whole thing run as a github action.
